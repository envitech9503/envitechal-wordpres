# Premium Post Workflow

Use GeneratePress plus GenerateBlocks for complex Envi Tech AL service and SEO posts.

## Rules

- Do not paste ChatGPT output as raw HTML into post content.
- Do not paste Markdown code fences, script tags, style tags, or JSON-LD scripts into post content.
- Build premium posts from the pattern inserter: `Envi Tech AL Premium Posts`.
- Use the full shell pattern first, then edit text, cards, links, FAQ answers, and table rows in the block editor.
- Keep layout classes in GenerateBlocks blocks or the child theme stylesheet.
- Keep FAQ schema in Rank Math. Do not add inline FAQPage script tags inside post content.
- Preview the public frontend after publishing, not only the editor preview.

## Recommended Build Order

1. Insert `ETA Premium Full Post - Water Testing` or the closest full shell pattern.
2. Replace the hero title, lead, buttons, and service links.
3. Edit stats cards and trust bar signals.
4. Update risk alert and two-column explanation content.
5. Replace parameter cards with service-specific checks.
6. Edit the comparison table as a real table block.
7. Update workflow steps.
8. Add internal links and authoritative external references.
9. Create FAQs using the FAQ accordion pattern, then enter matching FAQ schema in Rank Math.
10. Update the final CTA.
11. Confirm public source and rendered page have no triple backticks, no Markdown headings, and no flattened table text.
